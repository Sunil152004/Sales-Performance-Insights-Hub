# Sales Performance Insights Hub

## Project Overview
Sales Performance Insights Hub is an Excel-based Business Intelligence dashboard project created to analyze and visualize retail business data.

This project provides interactive dashboards to understand sales performance, customer behavior, inventory status, and business trends.

## Dashboards Included
- Executive Dashboard
- Sales Dashboard
- Customer Dashboard
- Inventory Dashboard
- Forecast Dashboard

## Key Insights Covered
- Sales and revenue analysis
- Customer performance analysis
- Product and inventory tracking
- Business KPIs monitoring
- Sales trends and forecasting

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Dashboard Design

## Project Objective
The main goal of this project is to transform raw retail data into meaningful insights that support data-driven business decisions.
